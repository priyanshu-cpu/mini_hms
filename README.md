# Hospital Management System

A simple Django-based web application for managing doctor–patient appointments.

---

##  Features

-  User Authentication (Register, Login, Logout)
-  Role-based system:
  - Patient
  - Doctor
-  Appointment Booking (Patient → Doctor)
-  Doctor Dashboard (view upcoming appointments)
-  Access Control (patients and doctors have separate actions)

---

##  Tech Stack

- Python
- Django
- HTML, CSS (Bootstrap)
